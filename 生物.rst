試劑
====
氯化亞鈷 ------> 水

                          可逆
    無水硫酸銅(藍)   + 水 <--> 含水硫酸銅(紅) + 熱 
    無水氯化亞鈷(藍) + 水 <--> 含水氯化亞鈷(紫紅) + 熱 

石蕊試紙 ------> PH
蝶豆花 --------> PH, 酸 ------> 紫紅
                     小蘇打 --> 藍綠色
                     加熱 ----> 不變色 (藍)
                     (酸) 紅 ---> 紫紅 ---> 粉紅 ---> 藍(中性) ---> 藍綠 ---> 黃 (鹼性)

廣用試劑/廣用試紙 --> PH (顏色與彩虹一樣)
                     紅(PH0) ---> 橘(PH1) --> 黃(PH4) ---綠(PH7) --> 藍 (PH10) --> 紫 (PH14)

酚酞指示劑 ----------> PH <= 7 無色
                       PH > 7 ------> 漸漸變成深粉紅 
                       
PH計 ---------------> 測離子濃度, 直接顯示 PH 值的數值

化學式
======

酸 + 鹼 = 鹽類 + 水 + 熱

鹽酸 (HCl) + 氫氧化鈉 (NaOH) ----> NaCl (食鹽) + H2O + 熱 (heat)

鹽類
====
NaCl ..... 氯化鈉, 食鹽
CaSO4 .... 硫酸鈣, 石膏
CaCO3 .... 碳酸鈣, 貝殼, 大理石, 鈣片
NaHCO3 ... 碳酸氫鈉, 小蘇打 (焙用鹼)
Na2CO3 ... 碳酸鈉, 蘇打, 用於清潔劑 (洗滌鹼)
